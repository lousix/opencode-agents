## 一、严重度评估框架

### 评分模型

```
严重等级 = f(可达性, 影响范围, 利用复杂度)

评分公式:
score = a + b + c + d

a. 可达性: 未认证可达(+2) / 低权限(+1) / 管理员(+0)
b. 影响: RCE或全库(+3) / 部分数据(+2) / 信息收集(+1)
c. 利用复杂度: 单请求(+0) / 多步骤(-1) / 特定环境(-2)
d. 防护绕过: 无防护(+0) / 可绕过(+0) / 需额外条件(-1)

等级判定:
score ≥ 5 = Critical
score 3-4 = High
score 1-2 = Medium
score ≤ 0 = Low
```
